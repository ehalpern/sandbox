Intro to Frame

Frame is a lightweight scala framework for building production ready REST like services. It orchestrates a core of widely-used, high-quality libraries for configuration (Typesafe Config), modular construction (Google Guice), resilient concurrency and distribution (Akka) and asynchronous HTTP services (Spray).

The goal is to provide a minimal (yet sufficient) integration between these components so that they work well together but are not hidden. If you are using Frame you are using Spray and Akka. Frame attempts to solve the little problems around the edges like how to manage external configuration, how to maintain a modular codebase, simplifying the testing of akka and spray routing services, etc.
